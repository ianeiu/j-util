使用频率极高的快捷键：
代码提示	 			alt+/
自动生成object方法		alt + shift + s，然后根据提示继续
复制当前行				ctrl +alt + ↓
 查找文本				ctrl + k ,ctrl + shfit + k
 如何跳转到文件中的特定行	ctrl + l
调出搜索文本的视图		ctrl + f
进行全局搜索和局部搜索		ctrl + h


类：
1   查看接口实现类	ctrl + t
2 查看下一个方法  ctrl + shift + ↓
3   找出调用某个方法的所有类	ctrl + alt + h

视图：
1 文件切换的三种方式
	1.1 显示出当前打开的所有文件	ctrl + e
	1.2 文件切换		ctrl + pg、up ,ctrl + pg、dn
2  视图切换				ctrl + f7	跳转到代码编辑器视图 f12
3   经典的quick access	ctrl + 3  搜maven、server
4 eclipse的面包屑功能	alt + shift + b

文件：
1   创建各种类型的文件	ctrl + n
2   关闭当前文件		ctrl + w
3   关闭所有文件		ctrl + shift + w
3   打开文件所在目录	alt + shift + w，选择system explorer

文本：
1  删除当前行				ctrl + d
2  查看字符串出现的地方			ctrl + shift + u
3 显示html和xml元素的所有属性	ctrl + shift + 逗号
4  大小写转换				变小写ctrl + shift + y,变大写ctrl +shift + x。
5 重构
	5.1  修改名字 			alt + shift + r 统一修改方法的参数名字、或者类变量、方法变量
	5.2  抽取类方法			选中代码，然后使用alt + shift + m,就会弹出抽取视图
