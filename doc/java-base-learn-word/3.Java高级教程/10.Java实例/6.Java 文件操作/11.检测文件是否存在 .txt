以下实例演示了使用 File 类的 file.exists() 方法来检测文件是否存在：

/*
 author by w3cschool.cc
 Main.java
 */

import java.io.File;

public class Main {
   public static void main(String[] args) {
      File file = new File("C:/java.txt");
      System.out.println(file.exists());
   }
}

以上代码运行输出结果为（如果你的 C 盘中存在文件 java.txt）：

true
