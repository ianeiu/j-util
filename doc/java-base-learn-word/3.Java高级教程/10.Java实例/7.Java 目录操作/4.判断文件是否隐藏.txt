以下实例演示了使用 File 类的 file.isHidden() 方法来判断文件是否隐藏：
/*
 author by w3cschool.cc
 Main.java
 */

import java.io.File;

public class Main {
   public static void main(String[] args) {
      File file = new File("C:/Demo.txt");
      System.out.println(file.isHidden());
   }
}
以上代码运行输出结果为（Demo.txt为隐藏文件）：
True
